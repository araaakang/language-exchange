# 🌏 Language Exchange Platform

一個以語言交換為主題的配對平台，幫助使用者找到適合的語伴，建立交流並開始對話。

## ✨ 功能

- Google 登入
- 編輯個人資料
- 設定母語與想學習的語言
- 選擇個人興趣標籤
- 根據語言與興趣篩選語伴
- 收藏感興趣的使用者
- 發送、接受或拒絕語伴邀請
- 查看其他使用者的個人資料

## 🛠️ Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Firebase Authentication
- Cloud Firestore

## 🚀 Getting Started

```bash
npm install
npm run dev
```

開啟瀏覽器：

```
http://localhost:3000
```

即可開始使用。
